# Start
- For run only on device
  - npm
    ``` shell
      npm start
      npm run develop
    ```
  - yarn 
    ``` shell
      yarn start
      yarn develop
    ```
  - gatsby 
    ``` shell
      gatsby develop
    ```
  
- For run in LAN
  - npm
    ``` shell
      npm run develop:e
    ```
  - yarn
    ``` shell
      yarn develop:e
    ```
  - gatsby
    ``` shell
      gatsby develop -H 0.0.0.0
    ```
